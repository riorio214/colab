2.소품샵의 정의에 의거해 관련 데이터 추출 후 데이터 csv를 따로 만들기

4.csv를 활용해 지역별 점포 개수를 찾기
5.지역별 분포도 및 차트 작성하기
6.각 지역별 지도 그리기
